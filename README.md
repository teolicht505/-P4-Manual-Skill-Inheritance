# Manual Skill Inheritance for P4 (PS2)
## Description
This mod allows players to manually select the skills their personas inherit during fusion, eliminating the need for skill rerolling and adding an extra layer of strategy to the game.

![](Manual_Inheritance/Preview.png)

## Compilation
This mod is written in a custom language for PS2 cheats called CLPS2C, so the first step to compile this mod is to use the [CLPS2C-Compiler](https://github.com/NiV-L-A/CLPS2C-Compiler/tree/master). The version used to compile this cheat is 1.0.5.
